=Windows=
    Run HermesInstaller.exe executable, preferably as administrator if your user doesn't have write privileges on PostgreSQL installation directory.

=Linux=
    Run HermesInstaller executable, preferably as root if your user doesn't have write privileges on PostgreSQL installation directory.

=OS X=
    Run HermesInstaller application, preferably as root if your user doesn't have write privileges on PostgreSQL installation directory.
